Kanban Board Task Management Application

Project Overview

This project is a Kanban Board Task Management Application developed using React.js and Tailwind CSS.
It helps users manage tasks using two sections:

* Todo
* Done

Users can add, move, and delete tasks dynamically.


Technologies Used

* React.js
* JSX
* JavaScript ES6
* Tailwind CSS
* Vite
* React Icons

 Features

* Add Tasks
* Delete Tasks
* Move Tasks between columns
* Dynamic Task Counter
* Responsive UI
* Hover Effects
* Empty State Message


Commands Used

npm create vite@latest
npm install
npm install tailwindcss @tailwindcss/vite
npm install react-icons
npm run dev


 React Concepts Used

* Components
* Props
* useState Hook
* Conditional Rendering
* Event Handling
* Dynamic Rendering
